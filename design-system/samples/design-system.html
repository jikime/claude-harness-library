14 · Known Gaps

미관측 / 한계

home 단일 페이지 1회 관측. 아래 컴포넌트는 DOM 미검출이라 전시하지 않고 라벨만 표기합니다(지어내지 않음).

미관측 표준 컴포넌트

dialog / modal toast tooltip popover tabs (role) table calendar / date select radio switch textarea

상태/마크 미캡처

button hover/active/disabled 결과 면색 · checkbox checked 커스텀 마크 · pulldown 열린 메뉴 · field error/유효성.

대비 트레이드오프 (#2)

button-primary 흰 텍스트 2.35:1 AA fail — 원본 충실 보존. AA 필요 시 button-primary-aa(ink 6.52:1). 코랄 텍스트는 흰 위 2.97:1로 대형만.

회색 발산 / primitive

회색 10종+ 혼재를 4단 잉크로 수렴(canonical은 다음 페이지 누적 시). primitive 색 램프 미노출(semantic 직행).

간격·높이·다크모드

section 96px 추정 · nav 높이 88px 채택(원본 computed 107px 병기) · 다크 테마 미노출(themes 그룹 생략) · 스크롤 키프레임 미계측.
